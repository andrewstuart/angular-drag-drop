ngDrop
directive in module uni

Description

Allows an expression to be evaluated upon drop. Event available as $event.

Usage

as attribute
<ANY ng-drop="{Expression}"
     ng-dragover="{Expression}"
     allow-drop="{String}">
   ...
</ANY>

Parameters

ParamTypeDetails
ngDropExpression

The expression to evaluate upon drop. If dropped element came from ngDrag, the ngDrag $scope is available as $from.

ngDragoverExpression

An expression to evaluate on dragover.

allowDropString

A string that identifies the type of data that will trigger the ngDrop handler. Used in conjuction with <ele ng-drag="..." ...></ele>, this is helpful for limiting what can be dropped on a particular target.

If the event has no data for the provided type, then the ng-drop function will not be called.
