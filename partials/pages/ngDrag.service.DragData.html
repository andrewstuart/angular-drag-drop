DragData
service in module ngDrag

Description

DragData is a service that holds references to $scopes for the drag directives. The source will be exposed for the ng-drop expression.

Methods

  • add(scope)

    add adds a scope to the service to be tracked by its $id.

    Parameters
    ParamTypeDetails
    scopeScope

    The scope to track

  • get(id)

    get returns the scope with the given $id.

    Parameters
    ParamTypeDetails
    idString

    The $id of the scope that you want returned.
