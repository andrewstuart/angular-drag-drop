ngDrag
directive in module angular-drag-drop

Description

A directive that allows an item to be dragged and its $scope tracked and provided to the drop target upon drop.

Usage

as attribute
<ANY ng-drag="{String}"
     ng-dragstart="{Expression}"
     ng-dragend="{Expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
ngDragString

Either emtpy or a string that can be used to identify the type of data that is being transferred. This is most useful for limiting where an item can be dropped. For example, say you show multiple items each with multiple steps, and you want to limit dragging steps to the same item.

<div class="item" ng-drag="item/{{item.id}}">...</div>
ngDragstartExpression

An expression that will be executed on dragstart. Event available as $event.

ngDragendExpression

An expression that will be evaluated on dragend. Event available as $event.
